단자음키
Manual

아이폰 정규 키보드가 불편하게 느껴본 적이 있다면 그것은 아마, 자소 결합으로 구성되는 한글의 특성을 제대로 반영하지 못하는 한계를 느끼기 때문일 것입니다
종성과 초성을 함께 사용하고 자소 결합을 특성으로 하는 한글을 온전히 사용할 수 있어야겠습니다

• '단자음키' 키보드에는 초성 또는 종성의 겹자음을 분리하겠다는 분명한 의지가 반영되는 키보드 규칙이 있습니다
-> 겹자소에 해당하는 키를 더블터치할 때 첫번째보다 두번째 터치를 좀더 짧게 터치하면 겹자소(쌍자음, 겹자음, 쌍모음)가 생성됩니다.


[ 기존 다른 키보드보다 개선된 점 ]
• 종성과 초성을 분리하기하거나 별도 자소 키 스킨을 바꾸기위해서 독립된 다른 버튼(예, 스페이스바)을 한번 더 누르지 않음
• 예) ‘ㄲ’을 만들기위해 'ㄱ' 키를 누른고 말 풍선 뜨기를 기다린후 드러그하여 ‘ㄲ’을 선택하는 과정 근원적 제거
• ‘학교’와 같이 동일 자음을 빠르게 입력하면 ‘하꾜’식의 오류를 근원적으로 제거
• 양손 멀티 터치 동작이 가능해 이를 적용함으로서 입력을 부드럽게 할 수 있었음
• 아이폰 정규 키 버튼 갯수를 1/3 가량 줄이게 됨으로서 상대적으로 키 버튼 크기가 커져, 터치시 오타(휴먼 애러)발생율 대폭 개선
• 표준 키보드의 중요 자음 키 배열을 대부분 동일하게 배치할 수 있었음

[ 추가 적용된 것 ]
• 아래 방향 쓸어내림 동작으로 단자소를 겹자소로 교정하는 편의 적용
• 위 방향 쓸어올림 동작으로 2번째 키(예, '?' or '1,2,3..'등) 빠르게 사용
• 타자 속도에 맞춰 겹자소 생성 누름 시간 간격 자동 조정
• 숫자 패널 나타내기 숨기기를 키보드 제스처로 처리함 (키보드에서 3칸 이상 아래로 쓸어 내리기)
