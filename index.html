Manual

아이폰 정규 키보드가 불편하게 느껴본 적이 있다면 그것은 키보드 버튼의 크기가 작아 발생하는 휴먼 오타 떄문일 것입니다
휴대폰의 제한된 크기를 감안한 더 나은 사용자 경험이 필요합니다

v 1.6.1
- 단어에서 자소 삭제 후 다시 연결 입력시 글자 구성이 계속 되도록 수정
- ㅐ : ㅏ + ㅣ, ㅙ: ㅘ + ㅣ, ㅔ : ㅓ + ㅣ, ㅞ : ㅝ + ㅣ 자소 생성 방법 추가
- 쌍자음 : '터치하여 우측으로 밀기' 기능 추가 (ㄲ ㅉ …)등
- 특수문자 : '터치하여 위 쪽으로 밀기' 기능 추가 (1,2…, ‘ . ‘ ? !)등
v 1.6
- 자음 키 배열 : 쿼티 키보드와 좀 더 유사한 배열로 채용함
- 모음 키 배열 : 단모음 키보드와 상당 부분 유사한 배열로 채용함
v 1.5.3
- 스패이스바 더블클릭으로 구듯점(' . ') 생성
- 스패이스바 좌.우로 움직여 입력 포인트(커서) 이동
- 기호 '하트'류를 심볼 페이지에 추가
v 1.5
- 경음(ㄲ,ㄸ,ㅃ...) > 홀드업 풍선에서 선택
- 격음(ㅋ,ㅍ,ㅌ, ㅕ,ㅛ...) > 자음 더블클릭, 모음 더블 or 연속클릭으로 생성
- 겹받침(ㄼ,ㄺ,ㅞ...) > 단자소 연속클릭으로 생성
- 아이폰 정규 키보드 자판에서 키 버튼의 갯수를 26 개에서 17 개로 35%를 줄이는 대신 격자소 더블클릭 기능을 추가하였습니다

[ 특징 ]
1. 터치에 적당한 크기로 하여 버튼의 개수를 정하였습니다
버튼이 너무 많으면 터치 수는 적으나 터치 실수 오타가 많고 너무 적으면 키인 수는 증가되나 터치 실수가 적어집니다
2. 최대한 빠른 키인을 할 수 있도록 하였습니다
버튼이 커짐으로 신속히 이동할 수 있어 원터치,더블터치,풍선말(ㅃ,ㅉ,ㄸ,ㄲ)을 이용하는 전반적 속도가 빠릅니다
3. 더블터치류의 자소로 인한 오타(ㅋ -> ㄱㄱ)를 교정하는 자동 강제교정 및 국어사전(국립국어원 표준기초사전)을 참고하는 자동 사전교정으로 처리합니다
4. 양손 엄지를 동시에 누르며 키인을 이어가는 멀티터치가 완전하게 구현되어 있습니다 내부적으로 자동교정이 진행되어도 정상 처리됩니다
